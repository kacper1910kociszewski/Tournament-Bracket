    # Tournament Bracket Builder 2.5

## Project Overview
Real-time tournament bracket builder with role-based permissions, built with:
- React (SPA)
- React Router
- Material UI
- Firebase (Auth + Firestore)

**Features planned:**
- User registration & login
- Create tournament & add players
- Randomize player order
- Real-time bracket updates
- Admin/owner roles
- Invite system
- Export brackets (stretch)
